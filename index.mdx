# Build Your Own Stunning Portfolio

**Psi Lime Portfolio** is a modern, production-ready React portfolio template built with cutting-edge web technologies. Showcase your work with beautiful 3D graphics, smooth animations, and interactive components that leave a lasting impression.

## Key Features

<CardGroup cols={2}>
  <Card title="WebGL Aurora Background" icon="wand-magic-sparkles" href="/components/aurora">
    Custom shader-based aurora effect with configurable colors and animations
  </Card>
  <Card title="Interactive Components" icon="hand-pointer" href="/components/spotlight-card">
    Mouse-tracking spotlight cards and infinite scrolling logo carousels
  </Card>
  <Card title="3D Graphics Ready" icon="cube" href="/quickstart">
    Three.js integration with react-three-fiber for stunning 3D visuals
  </Card>
  <Card title="Contact Form" icon="envelope" href="/components/contact">
    EmailJS integration with reCAPTCHA protection for secure communication
  </Card>
</CardGroup>

## Tech Stack

Built with modern tools and frameworks:

- **React 19** - Latest React with concurrent features
- **Vite** - Lightning-fast build tool and dev server
- **TailwindCSS** - Utility-first CSS framework
- **Three.js** - 3D graphics and WebGL rendering
- **GSAP** - Professional-grade animation library
- **OGL** - Lightweight WebGL library for custom shaders

## Quick Start

Get up and running in minutes:

<Steps>
  <Step title="Clone the Repository">
    Clone the portfolio template to your local machine:
    ```bash
    git clone https://github.com/JDiazRzo/Portfolio.git
    cd Portfolio
    ```
  </Step>

  <Step title="Install Dependencies">
    Install all required packages using your preferred package manager:
    <CodeGroup>
    ```bash npm
    npm install
    ```
    ```bash yarn
    yarn install
    ```
    ```bash pnpm
    pnpm install
    ```
    </CodeGroup>
  </Step>

  <Step title="Start Development Server">
    Launch the development server with hot module replacement:
    ```bash
    npm run dev
    ```
    Your portfolio will be available at `http://localhost:5173`
  </Step>
</Steps>

## What's Inside

This portfolio template includes everything you need to create a professional online presence:

### Presentation Section
A hero section with your profile picture, bio, location, availability status, and downloadable CV. Includes social media links to GitHub, LinkedIn, and Instagram.

### About Me
Showcase your background, skills, and experience with an animated technology stack carousel featuring logos of tools you know.

### Projects Showcase
Display your best work with interactive project cards, including live demo links and source code repositories.

### Contact Form
Engage with visitors through a beautiful contact form powered by EmailJS, complete with reCAPTCHA verification for spam protection.

### Custom Visual Effects
- **Aurora Background**: WebGL-powered aurora borealis effect with customizable colors
- **Spotlight Cards**: Mouse-tracking spotlight effect on interactive cards
- **Logo Loop**: Infinite scrolling carousel for technology logos
- **Smooth Animations**: GSAP-powered transitions and effects
